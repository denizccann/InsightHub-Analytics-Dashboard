# InsightHub: Corporate Data Analytics Dashboard 📈

### **Executive Summary**
InsightHub is a high-performance, web-based analytics engine designed to transform raw datasets into interactive business intelligence. This project demonstrates the integration of **Data Engineering** (Pandas) with **Modern UI/UX** (Streamlit), providing stakeholders with real-time KPI visibility.

### **Core Capabilities**
* **Interactive Visualization:** Dynamic Plotly charts for departmental productivity and budget correlation.
* **KPI Monitoring:** Real-time tracking of productivity scores, headcount, and system security status.
* **Audit-Ready Views:** Integrated data tables for direct verification of processed corporate records.
* **Enterprise UI:** Professional-grade "Dark Mode" dashboard optimized for executive reporting.

### **Technical Architecture**
* **Framework:** Streamlit (Web UI)
* **Visual Engine:** Plotly (Interactive Charts)
* **Data Processing:** Pandas (Data Transformation)
* **Design Pattern:** Modular Pythonic Architecture
